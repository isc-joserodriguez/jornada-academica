Tendencias en el desarrollo web

En esta conferencia nos hablaron sobre las partes que integran la web.
Básicamente, tenemos 2 partes:

  • Front-end
    • HTML
    • CSS
    • JS
  • Back-end
    • DB
    • BL
    • WS

También existe 2 tipos de aplicaciones web
  • Progresive web Apps
  • Single Page Applications

Otra cosa que mencionó, fueron los frameworks. Los cuales trabajan con patrones de diseño.
Nos decía que él no recomienda unos sobre otros ya que el que más se adapte a lo que se va a trabajar, va a ser el mejor en ese caso particular.
Para terminar, nos dió algunos tips:
  1. Elegir un lenguaje y hacerlo tu primer amor.
  2. Conocer las estructuras de datos.
  3. Entrenar habilidades para crear algoritmos.
  4. Usar frameworks y comocer cómo funciona internamente.
  5. Usar GIT y GitHub.
  6. Aprovecha el potencial de los proyectos escolares.
  7. No esperar el último semestre para obtener experiencia.
  8. No sólo quejarse, actuar.
  9. Intentar hacer un negocio propio.
  10. Leer mucho.
