Investigación de Carrusel.

¿Qué es?


Un carrusel es un componente que se utiliza para recorrer elementos como si se tratara literalmente de un carrusel. En este caso veremos a continuación un ejemplo con imágenes combinado con texto:


Esta es la segunda imagen

¡Es una foto hermosa!

¡Hola!


Este es


un ejemplo


con texto.

Anterior Siguiente

Para crear este componente en específico tenemos el siguiente código:


<div id="myCarousel" class="carousel slide" data-ride="carousel" data-interval="5000" align="center">
----<div class="carousel-inner" role="listbox">
--------<div class="carousel-item active">
------------<img class="d-block img-fluid" src="img/imgEjemplo1.jpg">
--------</div>
--------<div class="carousel-item">
------------<img class="d-block img-fluid" src="img/imgEjemplo2.jpg">
------------<div class="carousel-caption">
----------------<h3>Esta es la segunda imagen</h3>
----------------<p>¡Es una foto hermosa!</p>
------------</div>
--------</div>
--------<div class="carousel-item">
------------<h1>¡Hola!</h1>
------------<br><h1>Este es</h1>
------------<br><h1>un ejemplo</h1>
------------<br><h1>con texto</h1>
--------</div>
----</div>
----<a class="carousel-control-prev" href="#myCarousel" role="button" data-slide="prev">
--------<span class="carousel-control-prev-icon" aria-hidden="true"></span>
--------<span class="sr-only">Anterior</span>
----</a>
----<a class="carousel-control-next" href="#myCarousel" role="button" data-slide="next">
--------<span class="carousel-control-next-icon" aria-hidden="true"></span>
--------<span class="sr-only">Siguiente</span>
----</a>
</div>

Como podemos apreciar, con "carousel-control-prev" y "carousel-control-next" controlamos la transición manualmente con la propiedad de href con el ID del carrusel.

Así mismo, podemos poner una transición automática con data-interval donde podemos ingresar el periodo en milisegundos en que se harán las transiciones.
